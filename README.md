# Kitchen-Timer
• SPA that allows user to set the timer<br>
• Enhanced user experience by adding GSAP animations<br>
• Added music when the timer runs out, and button to stop the music<br>
• Project built with UI/UX and responsive design principles <br>

## Tools used
 Written in vanilla JavaScript, HTML/CSS.
